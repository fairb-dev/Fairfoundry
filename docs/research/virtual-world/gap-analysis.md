| Micro (<$50M revenue / <50 employees) | 0 | 2 (Factory 23 Detroit Alloy 180 emp/$32M, Factory 17 HCMC 550 emp/$35M) | YES -- no true micro factory (<50 employees) |

### Specific Gaps

- **No mega-OEM** (>$50B): Companies like major auto OEMs, global mining conglomerates, or global consumer goods companies with massive supply chains are not represented.
- **No micro factory** (<50 employees): The smallest factory (Factory 16, Nagoya Seimitsu, 280 employees) is still mid-sized. No family-run workshops, single-CNC job shops, or artisan manufacturers.
- **No startup OEM**: OEM-04 (Greenfield, $420M) is described as startup-mentality but is already mid-size.

## 6. Test Type Gaps

### Test Types Covered

The 50 test scenarios cover: optical testing (MTF, SFR, luminance, color), electrical testing (AOI, SPI, ICT, continuity, hipot), mechanical testing (tensile, hardness, impact, fatigue, pressure), chemical testing (composition, dissolution, pH, purity), dimensional inspection (CMM, surface roughness), environmental (thermal cycling, salt spray), battery testing (capacity, impedance, K-value), RF testing (VSWR, S-parameters).

### Test Types Not Covered

| Test Type | Relevant Industries | Standards |
|---|---|---|
| **HALT / HASS** (Highly Accelerated Life Test / Highly Accelerated Stress Screening) | Electronics, automotive, aerospace | No standard (proprietary methodology) |
| **EMC / EMI testing** | All electronics, automotive, medical | IEC 61000 series, CISPR, MIL-STD-461 |
| **Vibration testing** (random, sine, shock) | Aerospace, automotive, defense, telecom | MIL-STD-810, ITTC 7.5, IEC 60068 |
| **Thermal shock** (rapid temperature transition) | Electronics, automotive | MIL-STD-883 Method 1011, IEC 60068-2-14 |
| **IP rating / ingress protection testing** | Consumer electronics, outdoor equipment | IEC 60529 (referenced but no dedicated scenario) |
| **Flammability / fire testing** | Construction, automotive, aerospace, textiles | UL 94, FMVSS 302, FAR 25.853 |
| **Acoustic / NVH testing** | Automotive, consumer electronics | SAE J2521 (brake squeal), ISO 3744 |
| **Leak detection** (helium, pressure decay) | Aerospace, automotive, pharma, HVAC | MIL-STD-883 Method 1014, ASTM E499 |
| **Cleanliness / particulate testing** | Automotive, medical, semiconductor | ISO 16232, VDA 19 |
| **Fatigue / durability** (long-duration) | Automotive, aerospace | ASTM E466, SAE J1099 |
| **Environmental stress screening (ESS)** | Defense, aerospace | MIL-HDBK-344 |
| **Flow / pressure testing** (hydraulic, pneumatic) | Oil & gas, automotive, HVAC | API 598, MSS SP-61 |
